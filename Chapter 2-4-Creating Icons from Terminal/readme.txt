To run these scripts:

1. create a directory
2. place a 1024x1024 px version of your icon inside that directory
3. copy iosIcons.sh and macaIcons.sh to same directory
4. launch terminal on the directory
5. type ./iosIcons.sh and press ENTER to run the iOS script or 
6. type ./macIcons and press ENTER to run the Mac script

All icons will be created and placed inside the correspondent directories.
